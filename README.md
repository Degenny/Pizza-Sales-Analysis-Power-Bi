# Pizza-Sales-Analysis-Power-Bi
SQl and Power-Bi analysis of pizza sales data to identify key business insights.
This project analyzes pizza sales data from a restaurant to answer business questions such as:
- Monthly order volume and seasonal patterns
- Most profitable day
- Peak order hours
- Customer preference by pizza size and category
- Top selling pizza types.
The analysis is performed using SQL for data extraction and Power BI for visualization.
Files:
Pizza Sales Business Analysis-SQL; SQL queries used to process and normalize the data
Power BI Visualization.pbix; Power BI dashboard
pizza_sales_dashboard.png; Screenshoot of the dashboard for quick view.
